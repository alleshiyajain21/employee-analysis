# employee-analysis
Employee Analysis 
________________________________________
Project Objective
The objective of this project is to analysis employee data using SQL to understand workforce distribution, salary patterns, departmental performance, and employee attrition indicators.
This project focuses on extracting meaningful HR insights to support data-driven workforce decisions.
________________________________________
Dataset Information
•	Source: Simulated HR dataset 
•	Tables Used:
o	employees
o	departments
o	salaries
o	job_roles
•	Key Columns:
o	Employee ID, Name, Age, Gender
o	Department, Job Role
o	Salary, Experience
o	Hire Date, Exit Date (if applicable)
________________________________________
Tools & Technologies Used
•	Database Language: SQL
•	Concepts Used:
o	SELECT, WHERE
o	GROUP BY, HAVING
o	ORDER BY
o	INNER JOIN, LEFT JOIN
o	Aggregate Functions (AVG, COUNT, SUM)
o	Subqueries
o	Date Functions
________________________________________
Data Preparation
•	Validated employee records and department mappings
•	Handled missing salary or role information
•	Ensured correct joins using primary and foreign keys
•	Standardized date formats for analysis
________________________________________
Analysis Performed
•	Analysis employee count by department and job role
•	Evaluated average and median salary by department
•	Studied experience and salary correlation
•	Identified high and low attrition departments
•	Analysis workforce distribution by age and gender
________________________________________
Key Insights
•	Certain departments had higher employee concentration
•	Salary distribution varied significantly across roles
•	Departments with higher workload showed higher attrition
•	Experience level influenced salary growth trends
________________________________________
Final Output
•	SQL query results showing:
o	Department-wise employee summary
o	Salary distribution analysis
o	Attrition and experience insights
•	Well-structured SQL scripts for analysis
________________________________________
Business Recommendations
•	Review compensation structure for low-performing departments
•	Focus retention strategies on high-attrition roles
•	Use workforce insights for better hiring and resource planning
•	Monitor salary equity across departments
